# dev-spring-boot

Spring Boot development environment with Docker Compose.

## Khởi chạy dự án

```bash
docker compose up -d --build
```

## Services

| Service | URL | Credentials |
|---------|-----|-------------|
| Spring Boot | http://localhost:8081 | - |
| Mongo Express | http://localhost:8082 | `admin` / `admin` |
